# C++ Project - Quadratic Root Calculator

A command line c++ application that calculates the roots of a quadratic equation.

## Usage

Simply enter the coefficients of the quadratic, `a`, `b` and `c`.

## Sample Output

```shell
Enter a: 6
Enter b: -7
Enter c: -3
Root 1 is 1.5
Root 2 is -0.333333
```

```shell
Enter a: 3
Enter b: -11
Enter c: -4
Root 1 is 4
Root 2 is -0.333333
```
